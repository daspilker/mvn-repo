mvn-repo
========

A Maven repository.

Compiling
---------

To compile and run this project you need [Java SE 7][1], [Maven 3.0][2] and [MongoDB][3]. When these
prerequisites are installed, simply type `mvn tomcat7:run` and open a browser at [http://localhost:8080/][4].
